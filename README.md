## GitHub Pagesで公開する手順

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内の `index.html`、`style.css`、`script.js` をアップロードします。
3. リポジトリの Settings → Pages を開きます。
4. Source を「Deploy from a branch」にします。
5. Branch を `main`、フォルダを `/root` にして保存します。
6. 数分後、表示されたURLからアプリを開けます。

## 修正したい場合

問題文、正答、語群を修正したい場合は `script.js` を編集してください。

- 〇×問題：`trueFalseQuestions`
- 問2の語群：`options`
- 問2の正答：`fillBlanks`

第32問は、問題文中の語句を「迎車回送」として登録しています。


## 動作確認用HTMLについて

`動作確認用.html` はCSSとJavaScriptを1つのHTML内にまとめた確認用ファイルです。GitHubへアップする前に、ブラウザで直接開いて動作を確認できます。公開時は通常どおり `index.html`、`style.css`、`script.js` をアップしてください。
